A simple and convenient calendar of events will not allow you to forget something important!

== Description ==
The plugin add a post type - event (date, name, description, tags). Events are added through the admin panel.

Date - datapicker, the remaining fields - standard.

For the frontend in the plugin make a calendar widget. After loading the page, it receives a list of events for the active month in the calendar and draws them; when the months are changed, the events are requested again.

== Installation ==

Install from a downloaded zip archive.

1. Go to your admin menu Plugins -> Add New, and click on "Upload Plugin" near the top.
2. Browse for the zip file and click on upload.

OR

1. Extract the downloaded zip archive in your wp-content/plugins directory.


Once uploaded,

1. Go to your "Installed Plugins" page and enable the plugin
2. Go to Settings -> Calendar of Events and choose where you want to have the search feature
